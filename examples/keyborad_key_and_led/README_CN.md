# 键盘按键与LED控制示例

## 程序功能说明

本示例程序演示了如何使用T-Display-SF32开发板控制键盘按键和LED灯。

## 硬件组件

- **XL9555**: IO扩展芯片
- **TCA8418**: 键盘扫描芯片
- **AW21009**: LED驱动芯片

## 程序运行现象

1. **初始化阶段**
   - 程序启动后，会初始化LOD_PIN (PA41) 引脚，设置为高电平
   - 初始化XL9555 IO扩展芯片
   - 使能WIFI模块 (XL9555_WIFI_EN_PIN引脚)
   - 使能键盘LED (XL9555_KEY_LED_EN_PIN引脚)
   - 初始化AW21009 LED驱动芯片
   - 初始化TCA8418键盘扫描芯片

2. **LED呼吸灯效果**
   - 创建一个名为"key_led_heart"的线程
   - 线程运行后，所有LED灯会呈现呼吸灯效果

3. **按键检测**
   - 主循环中不断检测键盘按键事件
   - 当检测到按键按下时，会在控制台输出按键码
   - 按键检测使用消息队列机制，实时响应按键事件

## 使用说明

1. 将程序烧录到T-Display-SF32开发板
2. 连接键盘和LED硬件
3. 重启开发板
4. 观察LED呼吸灯效果
5. 按下键盘按键，查看控制台输出的按键码

## 注意事项

- 确保硬件连接正确
- 确保键盘和LED驱动芯片已正确初始化
- 如果初始化失败，会在控制台输出相应的错误信息
