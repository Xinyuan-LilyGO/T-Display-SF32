# 📂 项目概述
    该项目基于RT-Thread操作系统，实现了多功能智能设备的用户界面和硬件外设管理。主要特性包括： 
# 🧩 核心功能模块
    用户界面系统 - 基于LVGL的图形界面
    电源管理 - BQ21080充电管理
    存储系统 - SD卡支持
    音频处理 - 音频编解码功能
    无线通信 - BLUETOOTH PAN共享网络实现
    蓝牙连接
    LoRa远程通信
    传感器系统
    BHI260AP或ICM20948 IMU传感器
    输入设备 - TCA8418键盘控制器
# ⚙️ 硬件要求
SF32LB52X微控制器
AMOLED(2.16英寸)显示屏:
    分辨率：240x240分辨率
    IC: CO5300
    TP IC:CST9220
支持的外设：
    SD卡接口
    蓝牙模块
    LoRa模块
    IMU传感器(BHI260AP或ICM20948)
    TCA8418键盘控制器

# ⚙️ 软件要求
RT-Thread
LVGL 图形库 (v9.2.0)
硬件驱动包:
  - tca8418 (键盘控制器)
  - bq21080 (电源管理)
  - bhi260ap/icm20948 (IMU传感器)
  - LoRa_radio_driver (LoRa通信)

# 📝 使用说明
    1. 开机启动: 上电开启蓝牙，第一次连接需手机上找到T-Dispaly-SF32并连接。若需要使用web配置，请打开手机的蓝牙共享网络功能,若初始化硬件失败，会弹出报错窗口。
    2. 主界面: 开机后，进入主界面，可触摸或者按键实现菜单的切换。
    3. 音乐界面：蓝牙音乐播放器，需连接蓝牙，支持歌曲（上/下）一首切换，暂停，播放，音量调节等功能
    4. LORA界面：可发送和接收LORA数据，可更改lora频率、功率、带宽等参数
    5. 传感器界面：可查看IMU传感器数据，可查看欧拉角等数据
    6. 充电界面：可查看电池电量，充电状态，充电电压，充电电流等数据
    7. 计时器界面：可设置倒计时，可暂停，继续等操作
    8. 天气界面：可查看天气，温度等数据，需连接蓝牙共享网络，获取天气数据
    9. 录音界面：可录音，播放录音
    10. 文件界面：可查看SD卡中的文件
    11. 键盘测试界面：可测试键盘功能
    12. 系统界面：可查看系统信息，可重启，关机等操作

### 按钮功能：
| Name | PIN   | 功能                           |
| ---- | ----- | ------------------------------ |
| A    | PIN34 | 开机                           |
| B    | PIN33 | 息屏与亮屏                     |
| C    | PIN35 | 无                             |
| D    | POWER | 进入shipmode开机键(长按5s以上) |


### 键盘按键映射：
|     | 1            | 2         | 3        | 4           |
| --- | ------------ | --------- | -------- | ----------- |
| 4   | KEY_PREV(⬅)  |           | KEY_HOME |             |
| 3   | KEY_ENTER(⏹) | KEY_UP(⬆) |          | KEY_DOWN(⬇) |
| 2   | KEY_NEXT(➡)  |           |          |             |
| 1   |              |           |          |             |
| 0   |              |           |          |             |

```c
KEY_PREV:选择上一项
KEY_ENTER:确认
KEY_NEXT:选择下一项
KEY_HOME:主界面
KEY_UP:带列表的元素，选择上一项
KEY_DOWN:带列表的元素，选择下一项
KEY_VOLUME_UP:音量增加
KEY_VOLUME_DOWN:音量减少
KEY_BL_UP:亮度增加
KEY_BL_DOWN:亮度减少
KEY_DROPDOWN：下拉菜单
```

# 📝 编译和烧录
编译烧录配置需要进入当前目录的project目录下，执行以下命令：
    1. scons --board=t-display-sf32_hcpu -j8       ---编译hcpu
    2. build_t-display-sf32_hcpu\uart_download.bat ---烧录hcpu
    3. scons --board=t-display-sf32 --menuconfig   ---配置菜单
如何安装环境，请看根目录下的readme文件
